AI 绘画 —— 模型的概念与分类

模型，是 AI 通过对图片的学习训练，然后整合生成的一个文件。如果我们喂给 AI 的图片都是二次元风格的，那这样得到的模型再怎么画也不可能画出真实风格；当然如果喂给 AI 的图片都是真实世界的照片，那 AI 也绝对画不出来二次元风格的图片。使用不同风格的模型，就能做出不同风格的作品。目前已经有许多的模型可供直接下载使用，不需要我们自己训练。

1. 模型的下载渠道
1.1 C站（需魔法，不收费）

  官网：https://civitai.com/

1.2 Hugging Face（需魔法，不收费）

  官网：https://huggingface.co/

1.3 Liblib AI（无需魔法，有收费倾向）

  官网：https://www.liblib.art/

2. 模型的分类

  模型分有大模型和小模型。大模型是 Checkpoint 模型，在绘画中必不可少；小模型包括 Hypernetwork 模型（画面微调）、Embeddings 模型（优化画风）、VAE 模型（调色滤镜）、Lora 模型（固定特定人物特征、优化画风），可根据自己需求随意添加。

2.1 Checkpoint 模型

  Checkpoint 模型，用来控制整体风格，有旧版的 SD 模型和新版的 SDXL 模型。

  现在的 Stable Diffusion 已经全面进入了 SDXL 的高分辨率时代。SDXL 的分辨率精度是过往底模的 4 倍，并且对提示词的理解程度有大幅提升，不用像 SD 模型输入一大段一大段的高精度前缀词，使我们可以把重心放在对主体的描述上。

  模型后缀通常是 .ckpt，占内存较大（3 ~ 10 GB），还有后缀为 .safetensors，占内存较小（1 ~ 2 GB）。模型文件的存放路径为 /models/ckpt。

  导入模型后，可在 SD 页面进行切换（如果没有，请刷新或重新启动 SD）。

2.2 VAE 模型

  VAE 模型，可以粗略的理解为 AI 绘画中的一种调色滤镜模型，主要影响画面的色彩质感。不过大多数 Checkpoint 模型，都已经把 VAE 整合到里面去了，并不需要我们再进行调用，当然如果使用了没有整合 VAE 的 Checkpoint 模型，作者一般也会注明哪种 VAE 配合效果最好。

  模型的后缀有 .pt 和 .safetensor，存放路径为 /autodl-tmp/models/VAE。

  导入模型后，可在 SD 页面进行切换。

2.3 Lora 模型

  Lora 模型，作用在于帮助我们向 AI 传达、描述某一个特征准确、主题清晰的形象。比如，我们向 AI 提出画一个“奥特曼”，但“奥特曼”是 AI 并没有接触过的概念，或者说，在 Checkpoint 模型中并没有“奥特曼”的信息，那么这时候，我们就可以借助 Lora 模型，在这样的 Lora 模型中，清晰的记录着“奥特曼”的各种特点和呈现方式，这样，AI 对于“奥特曼”的理解，一定更加全面且准确。不过，除了有控制人物特征的 Lora 外，还有控制风格的 Lora。

  Lora 模型也有 SD 版本和 SDXL 版本，一定要对照自己的 Checkpoint 模型版本使用，SD 版本的 Checkpoint 模型不可以使用 SDXL 版本的 Lora 模型，当然 SDXL 版本的 Checkpoint 模型也不可以使用 SD 版本的 Lora 模型。

  Lora 模型的存放路径为 /autodl-tmp/models/lora，占内存较小（几百 MB）。

  Lora 模型的使用是需要在提示词中调用的，书写规范为 <lora:xxx:权重>，xxx 为 Lora 的文件名，权重在 0 ~ 1 之间。

  当然提示词中最好也加上触发词（下载页面作者会注明）。

2.4 Embeddings 模型

  Embeddings 模型，全称是“嵌入式向量”。如果说 Checkpoint 模型是本大字典，那么 Embeddings 模型就是一个小书签，它能够精准的指向个别字词的含义。Embeddings 最主要的作用就是解决 AI 不会画手的问题，当然还有肢体混乱、颜色混杂、噪点和灰度异常等。例如，我们通常在绘画中为了防止生成多手多脚的人物，去加许多许多很难记住的反向提示词，而现在，可以直接在反向提示词中调用 Embeddings 模型，来解决这样的问题。

  Embeddings 模型的存放路径为 /autodl-tmp/models/embeddings。

  Embeddings 模型的使用方法与 Lora 模型相似，直接在提示词中调用即可（下载页面作者会注明触发词）。

2.5 Hypernetwork 模型

  Hypernetwork 模型一般被用于改善生成图像的画风，这个画风比 Checkpoint 所定义的画风要更为精细一些，并不是二次元和真实系之间的那种区别，而是像动漫卡通、书本插画、油画之间的区别。多数使用者对于 Hypernetwork 模型在图像生成方面的评价并不好，至少不如 Embeddings 和 Lora 模型，其次，它的作用已经被许多风格 Lora 模型所取代了。

3. 抄作业

  抄作业只需要 copy 该作品的所有参数即可，注意模型版本要一致（当然有的作者可能没有分享参数，有的可能也是错误的参数）。

  这个作品是有使用的 Embeddings 的，但相关链接并没有分享出来。
